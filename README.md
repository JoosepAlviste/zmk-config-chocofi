## Usage

1. Make a change
2. Push
3. Download firmware from the [GitHub Actions](https://github.com/JoosepAlviste/zmk-config-chocofi/actions) page
4. Double press the power button on the keyboard
5. Drag firmware onto keyboard. Only the left side is needed for keymap changes.
